2.2 变量

python中直接给变量复制, 变量会成为所赋的类型.比如message = “Hello Python!”为字符串类型.

  1. 变量命名

    1. 字母, 下划线, 数字, 开头不为数字
    2. 变量名不包含空格
    3. 不要用关键字命名

  2. 变量命名时避免犯错

    • 主要是拼写错误
    • 忘记赋值

2.3 字符串

可以用双引号”string” , 也可以是单引号’string’, 具体看字符串中字符包含哪一种, 那就使用另一种引号.(注意, 引号必须要有, 否则报错)

  1. 修改大小写

    • string.title(): 首字母大写
    • string.upper(): 全部大写
    • string.lower(): 全部小写

  2. 合并(拼接)

    • 使用加号+: string = string1 + string2

      注意: 可以在print中使用+, 如print(string1 + “ “ + strint2),其实string1 + “ “ + string2 就是字符串.

  3. 制表符/换行符的使用

    • \n: 换行
    • \t: 4个空格, 用于对齐数据

  4. 删除空白

    • string.strip(): 删除左侧和右侧的空白
    • string.lstrip(): 删除左侧所有空白
    • stirng.rstrip(): 删除右侧所有空白

      注意: 使用字符串得注意引号使用错误.
      python3中print()需要加括号, python2中不需要加.

2.4 数字

  1. 整数

    • 主要是各种运算+ - * / **(求次方)

  2. 浮点数

    • 带小数点的数字, 位数不定

  3. 数字转换为字符串

    • str(age)即可转换为字符串

  4. python2和python3中一些运算的区别

    • python2中/同C, 整数相除结果为整数.
    • python3中/有变化, 相处为小数, // 等价于上面的/.

  5. 添加注释

    • 单行注释: #
    • 多行注释: ‘’’ ‘’’ (3个引号)
    • 编写注释的目的是: 阐述代码要做什么, 如何做的.
