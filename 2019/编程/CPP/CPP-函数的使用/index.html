函数的使用

编程/ CPP
创建时间:2019-08-20 15:20

  2. 二.函数参数
  3. 三.函数返回值

一.函数命名

函数名称作为标识作用,最好通过名字就能知道函数的功能,有一定的命名规则

  1. 设置值的函数:setFunction(),常用于改变类私有变量的值的函数命名.
  2. 获得值的函数:getFunction(),常用于获得私有变量的值的函数命名.

补充:关于命名大小写的问题,总结如下:

  • 全大写:常量名,用_分割单词;
  • 全小写:变量名,函数名,用_分割单词;
  • 大小混合:即用单词首字母大写分割单词,用于变量名,函数名.

二.函数参数

分为两种情况,无参和有参.

  1. 对于无参函数:即调用时不需要外界传进参数,内部已有数据,常用于类的成员函数(因为成员函数是拥有自身对象的私有成员变量的数据的).也可以用于无需参数的某一种功能函数,比如输出图形(用的比较少)
  2. 对于有参函数:即调用时外界传进参数了,那么函数内部可使用的数据有函数已有的(比如私有成员变量)和外部传递进去的,该类函数一般需要对外界的数据进行处理可能会返回一定的值(不一定需要返回函数值,比如处理数据后,直接在函数内输出).

三.函数返回值

同样分为两种情况,无返回值和有返回值.

  1. 对于无返回值函数:其功能已在函数内部实现,不需要外界对其再进行处理,一般为功能类函数.
  2. 对于有返回值函数:此类函数一般是用于处理数据,再返回给外界,进行使用,一般为数据处理类函数.

四.函数参数和返回值类型

这里主要讨论两种有些许复杂的参数类型,& 和 const类型.

  1. &类型

    • 为何要使用&类型?通过直接传进去&类型的参数,函数调用传进去实参后,可直接对实参进行处理,而不是仅仅处理形参了,这对于需要修改数据时极为方便.但有人会说了,有时候我不想修改数据,那么只需要再类型前加上const即可避免修改数据,那有人可能又会说,为什么不直接使用Type name?这是因为引用传递效率高于将实参赋值给形参,养成一个好的习惯.
    • 如何使用&类型? int& x(需要改变数据时);const int& x(不想改变数据时,使用了引用传递效率更高,注意此时函数内部若是修改了数据将会报错).

  2. const类型的使用

    • const在函数参数中的使用:在&的使用中起到的作用如上,当然还不止时避免数据被修改,另一方面,可以增大函数传入实参种类的范围,还可以传递常量作为函数的实参(当然非常量也可),若是没加const则不能传递常量作为实参.
    • const在函数括号后面的使用,如void function() const;(此函数较为简单,无参数,无返回值).此const的作用是:可以避免对象的私有成员被修改,一旦修改对象的私有成员即会报错.因此,当我们只想访问私有成员变量而不是修改它时,可以加上const.
