类和对象以及非成员函数的理解

编程/ CPP
创建时间:2019-08-20 11:59

C++最大的不同于C语言的特点就是:面向对象编程(OOP).而在面向对象编程中,新的概念类和对象等比较抽象,难以理解,以下是我对其的一些理解和总结.

  1. 类

    • 什么是类?在我看来,类是一个集合的抽象,比如苹果/梨子/西瓜可以抽象为一个类,即水果.
    • 类有什么特点?类中抽象了这个集合的属性(即数据),行为(即方法/函数),比如对于一个水果类,可以抽象出:从属性看:size/shape/color等;从行为看(水果类可以干什么,有什么行为,外界对它的改变,称之为即方法):如cut.

  2. 对象

    在上述理解了类以及它的特点之后,那么对象就很容易理解了.

    • 什么是对象?对象是类的一个实例化,比如将水果类进行实例化,通过类定义一个对象Fruit fruit(_size, _shape, _color);通过构造函数赋予该对象具体的属性值,将其特殊化,有了这些具体属性值的对象就是一个具体的对象,我们可以知道它到底是苹果/梨子还是西瓜了.
    • 对象有什么特点?由于在类中的所有方法对于对象是通用的(因为抽象于对象),故对象可以使用类的方法.而且对象的属性只有通过自己类的方法才能访问,外界是不能访问的.

  3. 非成员函数

    在谈什么是非成员函数之前,明白什么是成员函数,顾名思义,作为成员的函数,类成员的函数.

    • 因此,不在类中定义的函数就是非成员函数,但是在同一个名字空间中的非成员函数一般与类的对象是有关系的,一般是以类的对象作为参数,对其进行一定的操作.
    • 特别注意一下,友元函数也是一种非成员函数,作为一种特殊的非成员函数.可以有成员函数的权限,访问私有成员变量.

  4. 友元函数

    对友元函数进行展开一下.

    • 什么是友元函数?友元函数是以friend声明的,其声明在类中,一般在public中.作为一种特殊的非成员函数.可以有成员函数的权限,访问私有成员变量.
    • 友元函数有什么作用?多用于运算符的重载,尤其是需要改变类对象和另一个数据的使用数据的运算符重载.比如cin<<fruit(若使用成员函数重载运算符,应当是fruit<<cin).

  5. 运算符重载

    运算符重载有两种方式,成员函数及非成员友元函数.

    • 成员函数重载:returnType operator +-/(otherType){};重载后的关系为stu +-/ x
    • 非成员函数重载:returnType operator =-/(otherType, stu){};重载后的顺序为括号里的顺序,因此可以随意更改顺序.比如*cin << Data** :就是一个重载<<运算符的例子.
    • 运算符重载后是有功能的,其功能一般保持原来的规则,但是使用对象会发生变化.
