一.Vim移动操作技巧(均在普通模式下)

1. 快速翻页

  • :下一页
  • :上一页

2. 同一页面快速跳跃

  • shirft-h:顶部
  • shirft-m:中部
  • shirft-l:底部

3. 快速跳行(通过查找)

  • /+内容:向下查找到相应位置
  • (shirft+/ = ?) + 内容: 向上查找到相应位置

4. 行内移动

(1)查找字母定位
  • f + 字母: 向后查找到相应字母处
  • F + 字母:向前查找到相应字母处
  • t + 字母: 向后查找到相应字母的前一个字母处
  • T + 字母: 向前查找到相应字母的前一个字母处
(3)特殊移动
  • 0 : 向行首移动, 索引为0的位置
  • $ : 向后末尾移动, 字符为截止符的位置
(2)单词移动
  • w: 向后移动一个单词(到下一个单词的第一个字符, 其他字符串为一个单词,比如({}))
  • b: 向前移动一个单词(到上一个单词的第一个字符, 其他字符串为一个单词,比如({}))
  • e: 首先移到当前单词的最后一个字符, 再向下一个单词的最后一个字符移动
  • W: 向后一个字符串的第一个字符移动(字符串为空格分隔)
  • B: 向前一个字符串的第一个字符移动
  • E: 向后一个字符串的最后一个字符移动

5. 定位返回

  • mm: 标记当前位置, 按`m 返回到标记的位置
  • `` : 返回到移动之前的位置
  • : 返回上一步; : 进一步
