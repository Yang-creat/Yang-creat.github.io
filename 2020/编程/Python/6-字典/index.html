6. 字典

6.1 使用字典

字典中的元素时由键-值对组成的, 其中键位字符串, 值可以为任意类型.

  1. 访问字典中的值: dic_name[‘key’] 方括号中加上键值即可访问对应元素
  2. 添加值: 通过赋值dic_name[‘new_key’] = new_value即可直接添加
  3. 创建空字典: dic_name = {}, 然后通过赋值添加元素
  4. 修改值: 赋值修改
  5. 删除值: del dic_name[‘key’] 类似于列表

6.2 遍历字典

  1. 遍历所有元素: dic_name.items()

  2. 遍历所有键: dic_name.keys() (默认为keys ,可以省略)

  3. 按顺序遍历所有键: sorted(dic_name.keys())

  4. 遍历字典中所有的值: dic_name.values()

6.3 嵌套

  1. 存放字典的列表: 即一个列表中存放多个字典, 有点类似于对象数组.
  2. 存放列表的字典: 当需要对某一个键由多个描述时, 用列表.
  3. 字典的字典: 相当于二级结构, 树状图

总而言之, 列表存放同一类型数据. 可以存放字典, 也可以作为字典中的值; 同时, 字典中的值, 可以为列表, 也可以为字典.
