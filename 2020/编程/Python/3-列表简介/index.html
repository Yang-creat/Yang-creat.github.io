3. 列表简介

3.1 什么是列表

  • 列表是由一系列特定顺序排列的元素组成.其中可以包含字母, 数字, 姓名等.
  • 通常列表命名通常使用小写字母,下划线_分隔, 并且使用复数,如letters, digits, names等.
  • 使用[ ]标识列表, 其中元素用 , 分隔.

  1. 访问列表元素

    • 直接通过下标访问即可, 同数组.例如bicycles[0],从0开始.

  2. 索引从0开始

    • 第一个元素下标为0
    • 最后一个元素下标为-1, 因此可以很方便地访问最后一个元素.

      注意列表索引越界.

3.2 增/删/改

  1. 增

    • 在尾部增加: list.append( )

      因此可以通过创建一个新的列表, 然而通过append()方法依次添加元素.

    • 在中间增加: list.insert(pos, string), 如list.insert.(0, ‘ducati’),插入到0位置, 其他元素后移.

  2. 删

    • 直接删除某个元素: del + name[i], 此处del为一个外部函数,并非list类的方法.
    • 删除元素并使用它:
      • list.pop(): 末位. bicycle = bicycles.pop()
      • list.pop(pos): 任意位置. bicycle = bicycles.pop(pos)

        以上两者的区别是在于删除后是否使用它.

    • 根据值删除元素: list.remove(‘string’) 直接删除列表中string字符串元素. 适用于不知道元素位置的情况.若需要保留删除的值, 可以在删除前将其赋值给一个变量保存, 然后将其删除. 如:

  3. 改

    • 类似于数组赋值, 直接给某个元素赋值即可. 如bicycles[0] = ‘ducati’.

3.3 组织列表

  1. 永久性排序

    • 正向排序: list.sort(),即可从小到大排序
    • 反向排序: list.sort(reverse = True), 即可从大到小排序

  2. 临时排序

    • 正向排序: sorted(list), 从小到大
    • 反向排序: sorted(list, reverse = Trur), 从大到小

      此函数是一个外部函数, 相当于创建一个临时列表.

  3. 逆序打印: list.reverse()即可逆转列表, 再次使用恢复原列表

  4. 列表长度: len(list)即可得到列表长度
