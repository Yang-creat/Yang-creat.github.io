9.类

9.1 创建和使用类

一. 创建类
  1. 定义类: 类名首字母大写
  2. 方法init()
    • 前后两个_
    • 其中必有形参self: __ init __ (self, name, age), self用于传入实例本身的引用, 使得实例可以访问类中的属性和方法. 意思是, 当创建实例时, 需要将实例与类相关联, 也就是使用self关联实例和类, 类中本来对self的操作也就是对实例的操作了; 类似C++中的构造函数. 同时另外两个形参也就是传入了实例的属性.
    • 类中的所有方法都是对self.name 进行操作了.
      二. 创建实例

      通过 object = Object(‘name’)即可创建一个实例

  3. 访问属性
    python中有点不同的是可以直接使用实例的变量,C++中需要调用方法才能对私有变量进行操作.
  4. 调用方法
    直接使用 实例名.方法名() 即可调用方法

9.2 使用类和实例

一. 给属性指定默认值

由于类中在__ init __ (self, name=’’, age) 中定义了形参, 在创建实例时需要给name 和 age赋值, 当然也可以使用默认值, 比如name 的默认值就是空的.

二. 修改属性的值
  1. 直接修改属性的值: 利用赋值的方法即可修改属性值, 不同于C++
  2. 通过方法修改属性的值: 类似于C++
  3. 通过方法对属性的值进行递增: 同2, 也就是方法对属性值进行修改

9.3 继承

可以从已有的类衍生出来, 成为子类, 原来的类成为父类

一. 子类的方法__ init __ () (继承父类的属性和方法)

也就是在定义init函数时有点变化, 父类的形参在子类中必须要有, 当然可以增加一些形参(属性), 通过super().init(make)即可将属性传入父类中, 从而使得子类有了父类的属性和方法

二. 子类新的属性和方法

在子类 __ init __ () 函数中增加新的属性, 可以为默认值; 随后定义的方法均为新的方法.

三. 重写父类已有的方法

直接对父类的方法重写即可.

四. 将实例用作属性

可以将某一个类的实例作为另一个类的属性, 相当于将大类拆分成很多小类

9.4 导入类

  1. 导入类
    通过from module_name import class1_name, class2_name, 即通过逗号分隔导入多个类.
  2. 导入整个模块
    直接import module_name, 需要通过module_name.class_name()来访问类
  3. 导入模块中所有类
    from module_name import *

以上的三种类的导入均与函数的导入相同, 要么是直接导入模块, 用点.进行访问; 要么是导入模块中特定的函数/类, 可以直接通过名字使用; 要么导入模块中所有的函数/类(不太推荐, 当然确定无冲突时可以使用).

9.5 类编码风格

  1. 类的命名不同于函数: 类第一个字母大写, 且使用驼峰命名法
  2. 实例的命名同一般变量, 使用小写格式, 用下划线分隔
  3. 类定义后, 通过文档字符串描述类的功能
  4. 使用一个空格分隔方法, 使用两个空格分隔类
  5. 同时导入标准库中的模块和自己编写的模块时, 先导入标准库中的模块, 再使用一个空行, 再导入自己的模块.
